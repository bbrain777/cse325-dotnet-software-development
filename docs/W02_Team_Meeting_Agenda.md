### 4. Selected Project Idea: Expense Tracker

Project summary:

Team 1 Expense Tracker is a .NET Blazor web application that helps users track spending, organize expenses by category, monitor budgets, and view simple spending summaries.

Target users:

- Students
- Families
- Small business owners

Main features:

- User registration and login
- Add, view, edit, and delete expenses
- Create and manage expense categories
- Track monthly budgets
- Filter expenses by date, category, or amount
- Dashboard showing spending totals
- Basic spending reports or charts

CRUD data:

- Expenses
- Categories
- Budgets
- User profile data

Authentication needs:

- Each user should log in and see only their own expenses, categories, and budgets.
