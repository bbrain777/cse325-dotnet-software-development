# CSE 325 - .NET Software Development

Course workspace for CSE 325 assignments, notes, practice code, and projects.

## Setup Checklist

- VS Code installed and updated
- C# extension installed
- C# Dev Kit extension installed
- Git installed
- Git user name and email configured
- .NET SDK installed
- GitHub account available for public course repositories
- Trello account available for project management activities

## Folder Guide

- `week01` through `week07` - weekly notes, exercises, and starter files
- `projects` - course projects and assignment repositories
- `practice` - small experiments and scratch .NET work
- `docs` - course notes, screenshots, planning files, and references

## Helpful Commands

```powershell
dotnet --version
dotnet --list-sdks
git --version
git config --global --list
```

## Git Habits

- Commit work frequently with clear messages.
- Keep each course project organized in its own folder or repository.
- Do not commit passwords, API keys, connection strings, or private files.
- Push important work to GitHub before assignment due dates.
